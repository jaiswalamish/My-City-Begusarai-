# My-City-Begusarai-
- An app which basically have all the informations related to a city Begusarai like shops, malls, doctors, hotels, schools etc.
<br />
- Used Firebase Firestore Database as database, Firebase Storage to store images and Firebase Authentication for login and Registration.
<br />
- Used Inbuilt Firebase Email Verification to verify user detail.
<br />
- Users can upload their business domain by providing required details.
<br />
- Included map for all domain to get exact location.
<br />
- Tech Stack Used: Java, Android Studio, Firebase, Google Map API
